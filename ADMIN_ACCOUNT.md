## 技术实现

### 数据库初始化
在 `src/db/database.ts` 中，`init()` 方法会自动调用 `initializeSuperAdmin()` 来创建超级管理员账户。

### 用户类型扩展
- 在 `User` 接口中添加了 `isAdmin?: boolean` 字段
- 数据库 schema 中添加了 `isAdmin` 字段
- API 响应中包含 `isAdmin` 状态

### 前端显示
在个人资料页面，如果用户是管理员，会显示紫色的 "🔑 超级管理员" 徽章。

## 注意事项

- 超级管理员账户在首次初始化数据库时自动创建
- 如果账户已存在，不会重复创建
- 密码为明文存储（仅用于演示，生产环境应使用加密）
- 管理员权限目前仅用于标识，可扩展更多权限控制功能
